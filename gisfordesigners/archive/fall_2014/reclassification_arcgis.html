Reclassification

Data

Download lake_raleigh.gdb, conversion.gdb, and reclassification.gdb from ClassStore

Reclassify the data for suitability

Add and symbolize the DEM 

In ArcMap
Click the 'Add Data' button and select 'lake_raleigh_dem'
In the Layer Manager
Right click on the 'lake_raleigh_dem' layer and select 'Layer Properties'
In 'Layer Properties -> Display'
Set 'Resampling during display' to 'Bilinear interpolation'
In 'Layer Properties -> Symbology'
Set 'Color ramp' to your choice of color ramps
Set 'Stretch type' to 'Percent clip'

Turn on the Spatial Analyst Extension 

In the Customize menu
Select 'Extensions…'
Check the 'Spatial Analyst'

Environment settings 

In ArcToolbox
Right click in the white space of the ArcToolbox pane and select 'Environments…'
Set 'Workspace -> Current Workspace' to 'conversion.gdb'
Set 'Workspace -> Scratch Workspace' to 'reclassification.gdb'

Slope

Reclassify the slope using the natural breaks method 

In ArcToolbox
Select 'Spatial Analyst Tools -> Reclass -> Reclassify'
Set 'Input raster' to 'lake_raleigh_slope'
Set 'Reclass field' to 'Value'
Set 'Reclassification -> Classify -> Classification Method' to 'Natural Breaks (Jencks)' with 10 Classes
Set 'Output raster' to 'reclass_naturalbreaks'

Symbolize 'reclass_slope_naturalbreaks' 

In the Layer Manager
Right click on the 'reclass_slope_naturalbreaks' layer and select 'Layer Properties'
In 'Layer Properties -> Symbology -> Color Scheme' choose the 'Slope' color scheme

Reclassify the slope using the quantile method 

In ArcToolbox
Select 'Spatial Analyst Tools -> Reclass -> Reclassify'
Set 'Input raster' to 'lake_raleigh_slope'
Set 'Reclass field' to 'Value'
Set 'Reclassification -> Classify -> Classification Method' to 'Quantile' with 10 Classes
Set 'Output raster' to 'reclass_slope_quantile'

Symbolize 'reclass_slope_quantile' 

In the Layer Manager
Right click on the 'reclass_slope_quantile' layer and select 'Layer Properties'
In 'Layer Properties -> Symbology -> Color Scheme' choose the 'Slope' color scheme

Floodplain

Reclassify the floodplain

on a scale from 1 to 10 where 1 is safe and 10 is risky 

In ArcToolbox
Select 'Spatial Analyst Tools -> Reclass -> Reclassify'
Set 'Input raster' to 'floodplain_2006'
Set 'Reclass field' to 'FLOODZONE'
In 'Reclassification'
Set 'AEFW' to '10'
Set '1pct annual chance flood' to '3'
Set 'AE' to '5'
Set 'NoData' to '1'
Set 'Output raster' to 'reclass_floodplain'

Soils

Reclassify soils by suitability for trails

on a scale from 1 to 100 1 is suitable and 100 is suitable 

In ArcToolbox
Select 'Spatial Analyst Tools -> Reclass -> Reclassify'
Set 'Input raster' to 'soils_2013'
Set 'Reclass field' to 'MUSYM'
In 'Reclassification'
Set values according to soils table for trail suitability
Set 'Output raster' to 'reclass_soils_for_trails'
