Hydrological Modeling

Compute hydrology from terrain data in GRASS

Compute streams, overland flow, and flooding from terrain data.

start GRASS 7

in startup pannel
set GIS Data Directory path to your datasets
for example, C:\Users\myname\grassdata
for Project location select nc_spf (North Carolina, State Plane, feet)
for Accessible mapset select lake_raleigh
click Start GRASS

add the DEM 

d.rast lake_raleigh_dem

add the lake 

d.vect map=lake_raleigh fcolor=none

set region with 3 foot resolution 

g.region rast=lake_raleigh_dem res=3

convert the lake from vector to raster 

v.to.rast input=lake_raleigh@lake_raleigh output=lake_raleigh use=cat

compute flow direction, flow accumulation and subwatersheds 

r.watershed lake_raleigh_dem thresh=10000 accum=accum_10K drain=draindir_10K basin=basin_10K

use map algebra to remove the lake 

r.mapcalc "accum_10k_lake = if( isnull( lake_raleigh ), accum_10k , null())"
r.colors map=accum_10k_lake raster=accum_10k
r.mapcalc "draindir_10k_lake = if( isnull( lake_raleigh ), draindir_10k , null())"
r.colors map=draindir_10k_lake raster=draindirm_10k
r.mapcalc "basin_10k_lake = if( isnull( lake_raleigh ), basin_10k , null())"
r.colors map=basin_10k_lake raster=basin_10k

extract more detailed streams from flow accumulation raster

run r.mapcalc from GUI: type r.mapcalc and ENTER

provide name of the resulting map in the relevant field

paste or type the expression into the expression field and Run 

r.mapcalc "streams = if(abs(accum_10K) > 100, 1, null())"

convert rasters to vectors 

r.to.vect -s input=basin_10K output=basin_10K type=area
r.thin input=streams output=streams_thinned
r.to.vect -s input=streams_thinned output=streams type=line

overland flow hydrologic simulation using path sampling method

r.sim.water

compute the partial derivatives of the DEM 

r.slope.aspect elevation=lake_raleigh_dem dx=lake_raleigh_dem_dx dy=lake_raleigh_dem_dy zfactor=3.28084

set resolution 

g.region res=10

run the overland flow hydrological simulation 

r.sim.water --overwrite elevation=lake_raleigh_dem dx=lake_raleigh_dem_dx dy=lake_raleigh_dem_dy rain_value=500 depth=depth nwalk=10000 niter=4

flooding

add the wake county flood plain map 

d.vect map=floodplain_2006 fcolor=none

create a flood map for a given water level and seed point 

r.lake elevation=lake_raleigh_dem water_level=280 lake=lake_280 coordinates=2094295.68641,733625.679405

create a flood map for a higher water level 

r.lake elevation=lake_raleigh_dem water_level=288 lake=lake_288 coordinates=2094295.68641,733625.679405
