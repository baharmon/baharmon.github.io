Data Acquisition

Import terrain data into ArcGIS

In ArcGIS import and reproject the National Elevation Dataset (NED) digital elevation model (DEM) to North Carolina State Plane feet. 

In ArcCatalog or ArcMap
Open ArcToolbox
Select 'Data Management -> Projections and Transformations -> Raster -> Project Raster'
Set 'Input Raster' to 'ned19_n36x00_w078x75_nc_statewide_2003.tif'
Set 'Raster Dataset' to 'lake_raleigh.gdb/ncsu_ned_dem'
Set 'Output Coordinate System' to 'NAD_1983_StatePlane_North_Carolina_FIPS_3200_Feet' (EPSG code: 2264)
Set 'Resampling Technique' to 'BILINEAR'
Click 'OK'
