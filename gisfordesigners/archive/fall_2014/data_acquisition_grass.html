Download and unzip the nc_spf location. Move it to your grassdata directory.

Import terrain data into GRASS

In GRASS GIS import and reproject the National Elevation Dataset (NED) digital elevation model (DEM) to North Carolina State Plane feet.

Map Reprojection in GRASS

start GRASS 7

in startup pannel
set GIS Data Directory path to your datasets
click 'Define new location -> Location Wizard'

in the Location Wizard
set GIS Data Directory path to your datasets
name Project Location as 'nc_ned'
name Location Title as 'nc_ned'
click 'Next'
select 'Read projection and datum terms from a georeferenced data file'
click 'Next'
browse and select 'ned19_n36x00_w078x75_nc_statewide_2003.tif'
click 'Next'
click 'Yes' to import data
name your new mapset 'user' and click 'OK'

in startup pannel
for Project location select nc_spf (North Carolina, State Plane, feet)
for Accessible mapset select lake_raleigh
click Start GRASS

use r.proj to reproject the DEM to the current mapset in North State Plane feet. 

r.proj input=ned19_n36x00_w078x75_nc_statewide_2003 location=nc_ned mapset=PERMANENT output=ned_dem method=bilinear

add the DEM 

d.rast ned_dem

Import a Web Mapping Service in GRASS

In GRASS GIS import orthoimagery for your study area from a web mapping service (WMS).

start GRASS 7

in startup pannel
set GIS Data Directory path to your datasets
for example, C:\Users\myname\grassdata
for Project location select nc_spf (North Carolina, State Plane, feet)
for Accessible mapset select lake_raleigh
click Start GRASS

add the DEM 

d.rast lake_raleigh_dem

set region with 1 foot resolution 
g.region rast=lake_raleigh_dem res=1

add web mapping service with ortho imagery 
r.in.wms url=http://isse.cr.usgs.gov/arcgis/services/Orthoimagery/USGS_EROS_Ortho_1Foot/ImageServer/WMSServer output=usgs_eros_ortho layers=USGS_EROS_Ortho_1Foot

add a web mapping service for open street map 
r.in.wms url=http://irs.gis-lab.info layers=osm output=osm format=png
