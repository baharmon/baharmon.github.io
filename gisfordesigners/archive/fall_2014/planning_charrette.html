Trail Planning Charrette

Geovisualization Lab, Jordan 5111

9:00-11:30am

Essential supplies

12" or 18" tracing paper, 24" or 36" tracing paper, markers, pens, scales, masking tape, an aerial photograph of your site, and a contour map of your site.

Project brief

In this charrette each team will develop a series of trail scenarios for Lake Raleigh Woods. Teams will design scenarios for differing and potentially conflicting stakeholder objectives such as accessibility, experience and adventure, and conservation. For each scenario teams will compute a trail network using map overlay analysis and iterative cost path analysis.

Workshop groups

  • Tangeoms group one, leader: Brendan Harmon
  • Tangeoms group two, leader: Anna Petrasova
  • GRASS GIS group, leader: Vaclav Petras
  • ArcGIS group, leader: Mohsen Ghiasi

Schedule

Ideation

Each team presents their trail scenario ideas to their workshop leader (10 min)

Diagram the map overlay logic model for each scenario (5 min)

Sketch a quick concept for each scenario (5 min each)

Scenario planning

For each scenario (30-45 min each):

  • Cost mapping: use map overlay analysis to compute a cost map
  • Identify trailheads and destinations: create points for each site
  • Generate a trail network: iteratively compute the least cost path between sites
  • Refine the trail network: revise the cost map, site locations, and cost path parameters
