Download a 3dm model of the Chancellor’s House

Visualize data in Rhino in 3D

Open the model of the Chancellor’s House 

In Rhino
File -> Open
Select 'chancellors_house.3dm'

Set the viewport to rendered mode 

Click on the Perspective viewport dropdown tab
Set the mode to 'Rendered' or 'Raytraced with Neon' (if installed)

Turn on the sun 

In the command line type 'sun'
In the 'Sun' dialog box
Check 'On'
Set 'Time' to 'Now'
Set 'Location' to 'Here'

Experiment and adjust the time

Render the scene 

In the command line type '_DocumentProperties'
In the Rhino Render tab choose your settings
For example set 'Resolution' to '640x480'
For example set 'DPI' to '72'
For example set 'Anti-aliasing' to 'Normal (x2)'
In the command line type 'Render'
