Terrain visualization

Data

Download lake_raleigh.gdb from ClassStore

Visualize data in ArcMap in 2D

Add and symbolize the DEM 

In ArcMap
Click the 'Add Data' button and select 'lake_raleigh_dem'
In the Layer Manager
Right click on the 'lake_raleigh_dem' layer and select 'Layer Properties'
In 'Layer Properties -> Display'
Set 'Resampling during display' to 'Bilinear interpolation'
In 'Layer Properties -> Symbology'
Set 'Color ramp' to your choice of color ramps
Set 'Stretch type' to 'Percent clip'

Turn on the 3D Analyst Extension 

In the Customize menu
Select 'Extensions…'
Check the '3D Analyst'

Environment settings 

In ArcToolbox
Right click in the white space of the ArcToolbox pane and select 'Environments…'
Set 'Workspace -> Current Workspace' to 'lake_raleigh.gdb'
Set 'Workspace -> Scratch Workspace' to 'lake_raleigh.gdb'

Compute contour lines 

In ArcToolbox
Select '3D Analyst Tools -> Raster Surface -> Contour'
Set 'Input Raster' to 'lake_raleigh_dem'
Set 'Output Polyline Feature' to 'lake_raleigh_contours_1ft'
Set 'Contour Interval' to '1'

Symbolize the contours 

In the Layer Manager
Right click on the 'lake_raleigh_contours_1ft' layer and select 'Layer Properties'
In 'Layer Properties -> Symbology -> Symbol'
Choose a color and line width

Compute the hillshade 

In ArcToolbox
Select '3D Analyst Tools -> Raster Surface -> Hillshade'
Set 'Input Raster' to 'lake_raleigh_dem' 
Set 'Output Raster' to 'lake_raleigh_hillshade'
Check 'Model Shadows'

Symbolize the hillshade 

In the Layer Manager
Drag 'lake_raleigh_hillshade' underneath 'lake_raleigh_dem'
Right click on the 'lake_raleigh_dem' layer and select 'Layer Properties'
In 'Layer Properties -> Display'
Set 'Transparency' to '15%'

Export map 

File -> Export map...

Visualize data in ArcScene in 2.5D

Add and symbolize the DEM 

In ArcScene
Click the 'Add Data' button and select 'lake_raleigh_dem'
In the Layer Manager
Right click on the lake_raleigh_dem layer and select Layer properties
In 'Layer properties -> Display'
Set 'Resampling during display' to 'Bilinear Interpolation'
In 'Layer properties -> Symbology'
Set 'Color Ramp' to your choice of color ramps
Set 'Stretch type' to 'Percent Clip'
Optionally check 'Use hillshade effect'
In 'Layer properties -> Base Heights -> Elevation from Surface'
Set 'Floating on a custom surface' to 'lake_raleigh_dem'
In 'Layer properties -> Rendering'
Check 'Shade areal features relative to the scene’s light position'
Set 'Quality enhancement for raster images' to 'High'
In the Layer Manager
Right click on 'Scene Layers' and select 'Scene Properties'
'Scene Properties -> General'
Set 'Vertical exaggeration' to '2'

Add and symbolize contours 

Click the 'Add Data' button and select 'lake_raleigh_contours'
In the Layer Manager
Right click on the 'lake_raleigh_dem' layer and select 'Layer Properties'
In 'Layer Properties -> Base heights -> Elevation from features -> Use a constant value or expression'
In the Expression builder double click '[Contour]'
In 'Layer Properties -> Base heights -> Elevation from features'
Set 'Layer Offset' to '2'

Export scene 

File -> Export Scene -> 2D
