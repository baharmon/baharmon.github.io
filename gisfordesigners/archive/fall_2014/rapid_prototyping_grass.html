Rapid prototyping

Export xyz point clouds for 3d modeling and digital fabrication

Data

Download nc_spf from ClassStore or http://fatra.cnr.ncsu.edu/gisfordesigners/

Setup

start GRASS 7

in startup pannel
set GIS Data Directory path to your datasets
for example, C:\Users\myname\grassdata
for Project location select nc_spf (North Carolina, State Plane, feet)
for Accessible mapset select lake_raleigh
click Start GRASS

add the DEM 

d.rast lake_raleigh_dem@lake_raleigh

add the DSM 

d.rast lake_raleigh_dsm@lake_raleigh

set region with 3 foot resolution 

g.region n=734250 s=731750 e=2094750 w=2092250 res=3 save=lrwoods

Export xyz

export the digital elevation model to a space delimited text file with x, y, and z values 

r.out.xyz --overwrite input=lake_raleigh_dem@lake_raleigh output=dem.xyz separator=space

export the digital surface model to a space delimited text file with x, y, and z values 

r.out.xyz --overwrite input=lake_raleigh_dsm@lake_raleigh output=dsm.xyz separator=space

Export rasters

export the digital elevation model to a geotiff 

r.out.gdal input=lake_raleigh_dem@lake_raleigh output=dem.tif format=GTiff

export the digital surface model to a geotiff 

r.out.gdal input=lake_raleigh_dsm@lake_raleigh output=dsm.tif format=GTiff
