Terrain Modeling

Model and analyze terrain data in GRASS

Compute contours, slope, shaded relief, visibility, and solar irradiation from terrain data.

start GRASS 7

in startup pannel
set GIS Data Directory path to your datasets
for example, C:\Users\myname\grassdata
for Project location select nc_spf (North Carolina, State Plane, feet)
for Accessible mapset select lake_raleigh
click Start GRASS

add the DEM 

d.rast lake_raleigh_dem

set region with 3 foot resolution 

g.region rast=lake_raleigh_dem res=3

compute 3ft contours 

r.contour input=lake_raleigh_dem@lake_raleigh output=lake_raleigh_contours_3ft step=3

compute slope (in degrees) and aspect 

r.slope.aspect elevation=lake_raleigh_dem@lake_raleigh slope=lake_raleigh_slop aspect=lake_raleigh_aspect format=percent

add legend

compute shaded relief 

r.shaded.relief input=lake_raleigh_dem@lake_raleigh

to create an observer location

first query the map where you would like to create your observer point

copy the east and west coordinate values and the z value

for example: 2092159.22174,733003.189243,377.3769

paste the values (either comma or space delimited) into a text editor as save them as observer.txt

import points 

v.in.ascii -z input=observer.txt output=observer separator=, z=3

compute visibility 

r.viewshed -c -r input=lake_raleigh_dem@lake_raleigh output=lake_raleigh_viewshed coordinates=2092159.22174,733003.189243

compute solar irradiation 

r.sun elev_in=lake_raleigh_dem@lake_raleigh asp_in=lake_raleigh_aspect@lake_raleigh slope_in=lake_raleigh_slop@lake_raleigh beam_rad=bearm_radiance day=180
