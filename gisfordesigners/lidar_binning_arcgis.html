Lidar processing

Create a DSM and DEM from lidar

In either ArcMap or ArcCatalog

Create a LAS dataset

In ArcToolbox
Select 'Data Management Tools -> LAS Dataset -> Create LAS Dataset'
Set 'Input files' to '0793_009.las' through '0793_020.las'
Set 'Output LAS Dataset' to 'lake_raleigh.lasd'
Set 'Coordinate system' to 'Lambert_Conformal_Conic_2SP'
Check 'Compute statistics'

Create a mosaic dataset

In ArcToolbox
Select 'Data Management Tools -> Raster -> Mosaic Dataset -> Create Mosaic Dataset'
Set 'Output location' to 'lcc.gdb'
Set 'Mosaic Dataset name' to 'lake_raleigh_dsm'
Set 'Coordinate system' to 'Lambert_Conformal_Conic_2SP'
Set 'Pixel properties -> Pixel type' to '32_BIT_FLOAT'

Add the LAS dataset to the mosaic dataset to create the DSM

In ArcToolbox
Select 'Data Management Tools -> Raster -> Mosaic Dataset -> Add Rasters to Mosaic Dataset'
Set 'Mosaic Dataset' to 'lake_raleigh_dsm'
Set 'Raster type' to 'LAS Dataset'
Set 'Raster type properties -> Return type' to 'Any'
Set 'Raster type properties -> Class type' to 'Any'
Set 'Raster type properties -> Pixel size' to '3'
Set 'Raster type properties -> Cell aggregation type' to 'mean'
Set 'Raster type properties -> Void filling' to 'nearest neighbors'
Check 'Update cell size range'
Check 'Update boundary'
Check 'Update overviews'

Add the LAS dataset to the mosaic dataset to create the DEM

In ArcToolbox
Select 'Data Management Tools -> Raster -> Mosaic Dataset -> Add Rasters to Mosaic Dataset'
Set 'Mosaic Dataset' to 'lake_raleigh_dem'
Set 'Raster type' to 'LAS Dataset'
Set 'Raster type properties -> Return type' to 'Any'
Set 'Raster type properties -> Class type' to 'Ground'
Set 'Raster type properties -> Pixel size' to '3'
Set 'Raster type properties -> Cell aggregation type' to 'mean'
Set 'Raster type properties -> Void filling' to 'nearest neighbors'
Check 'Update cell size range'
Check 'Update boundary'
Check 'Update overviews'

Reproject the DSM mosaic to North Carolina State Plane feet

In ArcToolbox
Select 'Data Management -> Projections and Transformations -> Raster -> Project Raster'
Set 'Input raster' to 'lake_raleigh_dsm'
Set 'Output raster dataset' to 'lake_raleigh.gdb/lake_raleigh_dsm'
Set 'Output coordinate system to 'NAD_1983_StatePlane_North_Carolina_FIPS_3200_Feet'
Set 'Resampling technique to 'BILINEAR'

Symbolize 'lake_raleigh_dsm' 

In the Layer Manager
Right click on the 'reclass_slope_naturalbreaks' layer and select 'Layer Properties'
Set 'Layer Properties -> Display -> Resampling during display' to 'Bilinear interpolation'
Set 'Layer Properties -> Symbology -> Color Scheme' to the color scheme of your choice
Set 'Layer Properties -> Symbology -> Stretch Type' to 'histogram equalize' or 'standard deviations'

Reproject the DEM mosaic to North Carolina State Plane feet

In ArcToolbox
Select 'Data Management -> Projections and Transformations -> Raster -> Project Raster'
Set 'Input raster' to 'lake_raleigh_dem'
Set 'Output raster dataset' to 'lake_raleigh.gdb/lake_raleigh_dem'
Set 'Output coordinate system to 'NAD_1983_StatePlane_North_Carolina_FIPS_3200_Feet'
Set 'Resampling technique to 'BILINEAR'

Symbolize 'lake_raleigh_dem' 

In the Layer Manager
Right click on the 'reclass_slope_naturalbreaks' layer and select 'Layer Properties'
Set 'Layer Properties -> Display -> Resampling during display' to 'Bilinear interpolation'
Set 'Layer Properties -> Symbology -> Color Scheme' to the color scheme of your choice
Set 'Layer Properties -> Symbology -> Stretch Type' to 'percent clip'
