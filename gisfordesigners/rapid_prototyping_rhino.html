Rapid prototyping

Data

Download dem.xyz, dsm.xyz, dem.tif, and dsm.tif from ClassStore or http://fatra.cnr.ncsu.edu/gisfordesigners/

Compute terrain meshes from rasters

import the rasters into Rhino

import the raster digital elevation model (dem) as a mesh using the RhinoTerrain plugin 

run 'RhinoTerrain -> Import/Export -> Import raster elevation file'
or in the command line type 'RtImportElevation'
open 'dem.tif'
set 'Choose target coordinate system' to 'Use input data coordinate system'
set 'output type' to 'triangulated mesh'

import the raster digital surface model (dsm) as a mesh using the RhinoTerrain plugin 

run 'RhinoTerrain -> Import/Export -> Import raster elevation file'
or in the command line type 'RtImportElevation'
open 'dsm.tif'
set 'Choose target coordinate system' to 'Use input data coordinate system'
set 'output type' to 'triangulated mesh'

Compute terrain meshes from point clouds

Import the point clouds into Rhino

import the dem point cloud 

in the command line type 'import'
select 'dem.xyz'
set delimiters to 'Space'
check 'Create point cloud'

zoom all extents 

in the command line type 'z' for zoom
in the command line type 'a' for all
in the command line type 'e' for extents

assign the dem point cloud to a new layer 

rename 'layer 01' to 'dem point cloud'
change object layer
press 'esc' to clear the selection

zoom all extents 

in the command line type 'z' for zoom
in the command line type 'a' for all
in the command line type 'e' for extents

import the dsm point cloud 

in the command line type 'import'
select 'dsm.xyz'
set delimiters to 'Space'
check 'Create point cloud'

assign the dsm point cloud to a new layer 

rename 'layer 02' to 'dsm point cloud'
change object layer
press 'esc' to clear the selection

Compute terrain meshes

use the RhinoTerrain plugin to triangulate the dem point cloud into a mesh 

select the dem point cloud
run 'RhinoTerrain -> Mesh -> Create terrain mesh'
or in the command line type 'RtMeshTerrainCreate'
in the command line type 'a' for accept

assign the dem mesh to a new layer 

rename 'layer 03' to 'dem mesh'
change object layer
press 'esc' to clear the selection

use the RhinoTerrain plugin to triangulate the dem point cloud into a mesh 

select the dsm point cloud
run 'RhinoTerrain -> Mesh -> Create terrain mesh'
or in the command line type 'RtMeshTerrainCreate'
in the command line type 'a' for accept

assign the dsm mesh to a new layer 

rename 'layer 03' to 'dsm mesh'
change object layer
press 'esc' to clear the selection

Visualization

Visualize

set the viewport to rendered mode
click on the Perspective viewport dropdown tab
set the mode to 'Rendered' or 'Raytraced with Neon' (if installed)

Turn on the sun 

in the command line type 'sun'
in the 'Sun' dialog box
check 'On'
set 'Time' to 'Now'
set 'Location' to 'Here'

Preparing the model

Vertical exaggeration

scale the dem by a factor of 2 in the z-axis 

select the dem
in the command line type 'Scale1D'
set the 'Origin point' to '0,0,0' 
set the 'Scale factor' to '2'
set the 'Scale direction' by drawing a line in the z-axis with ortho mode on
or use the gumball

scale the dsm by a factor of 2 in the z-axis 

select the dsm
in the command line type 'Scale1D'
Origin point: '0,0,0' 
Scale factor: '2'
Scale direction: draw a line in the z-axis with ortho mode on
or use the gumball

zoom all extents 

in the command line type 'z' for zoom
in the command line type 'a' for all
in the command line type 'e' for extents

Rescale to 1':100"

scale by 1/100 

select all
in the command line type 'Scale'
set the 'Scale factor' to '1/100'

set the units to inches 

in the command line type 'units'
set 'Model units' to 'Inches'
in the 'scale model by 12' dialog click 'No'

zoom all extents 

in the command line type 'z' for zoom
in the command line type 'a' for all
in the command line type 'e' for extents

Create an inverse of the canopy

rotate the dsm by 180 degrees on the z axis to create an inverse of the canopy 

in the command line type 'Rotate'
set the 'First reference point' by drawing a vertical line with ortho mode on
set the 'Second reference point' to '180'
