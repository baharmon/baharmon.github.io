Fuzzy membership

Data

Download lake_raleigh.gdb, conversion.gdb, reclassification.gdb, and membership.gdb from ClassStore

Fuzzy membership

Reclassify the data with fuzzy membership functions

Nota bene: sybmolize the fuzzy membership layers with the 'Histogram Equalize' stretch type

Add and symbolize the DEM 

In ArcMap
Click the 'Add Data' button and select 'lake_raleigh_dem'
In the Layer Manager
Right click on the 'lake_raleigh_dem' layer and select 'Layer Properties'
In 'Layer Properties -> Display'
Set 'Resampling during display' to 'Bilinear interpolation'
In 'Layer Properties -> Symbology'
Set 'Color ramp' to your choice of color ramps
Set 'Stretch type' to 'Percent clip' or 'Histogram Equalize'

Turn on the Spatial Analyst Extension 

In the Customize menu
Select 'Extensions…'
Check the 'Spatial Analyst'

Environment settings 

In ArcToolbox
Right click in the white space of the ArcToolbox pane and select 'Environments…'
Set 'Workspace -> Current Workspace' to 'reclassification.gdb'
Set 'Workspace -> Scratch Workspace' to 'membership.gdb'
Set 'Raster Storage -> Pyramid -> Pyramid resampling technique' to 'NEAREST'
Set 'Raster Storage -> Resampling Method' to 'NEAREST'

Slope

Compute the fuzzy membership for 'reclass_slope_naturalbreaks'

on a scale from 0 to 1 where 0 is accessible and 1 is inaccessible 

In ArcToolbox
Select 'Spatial Analyst Tools -> Overlay -> Fuzzy Membership'
Set 'Input raster' to 'reclass_slope_naturalbreaks'
Set 'Output raster' to 'fuzzy_slope_naturalbreaks'
Set 'Membership type' to 'Large'
Set 'Midpoint' to '5.5'
Set 'Spread' to '1'
