Lidar

Generate a digital elevation model (DEM) and digital surface model (DSM) from lidar data

Data

Download nc_spf and the las directory from ClassStore or http://fatra.cnr.ncsu.edu/gisfordesigners/

Preview

load the las tiles in plas.io in the chrome browser to explore the data

Setup

start GRASS 7

in startup pannel
set GIS Data Directory path to your datasets
for example, C:\Users\myname\grassdata
for Project location select nc_spf (North Carolina, State Plane, feet)
for Accessible mapset select lake_raleigh
click Start GRASS

set the region 

g.region res=3

set the mask 

r.mask raster=mask@lake_raleigh

Compute the digital elevation model (DEM)

Import lidar data

import las files using v.in.lidar

filter using lidar class 2 to extract the bare earth points 

v.in.lidar -t -o input=0793_009.las output=ground_0793_009 class_filter=2
v.in.lidar -t -o -b input=0793_010.las output=ground_0793_010 class_filter=2
v.in.lidar -t -o -b input=0793_011.las output=ground_0793_011 class_filter=2
v.in.lidar -t -o -b input=0793_013.las output=ground_0793_013 class_filter=2
v.in.lidar -t -o -b input=0793_014.las output=ground_0793_014 class_filter=2
v.in.lidar -t -o -b input=0793_015.las output=ground_0793_015 class_filter=2
v.in.lidar -t -o -b input=0793_017.las output=ground_0793_017 class_filter=2
v.in.lidar -t -o -b input=0793_018.las output=ground_0793_018 class_filter=2
v.in.lidar -t -o -b input=0793_019.las output=ground_0793_019 class_filter=2

list the imported lidar tiles 

g.list type=vect pattern=ground_*

combine the lidar tiles with v.patch 

v.patch -b --overwrite input=ground_0793_009,ground_0793_010,ground_0793_011,ground_0793_013,ground_0793_014,ground_0793_015,ground_0793_017,ground_0793_018,ground_0793_019 output=ground

Interpolate the DEM

compute the DEM using regularized spline with tension interpolation 

v.surf.rst -t --overwrite input=ground@lidar elevation=lake_raleigh_dem tension=100 npmin=250 dmin=2

compute the DEM using bilinear spline interpolation 

v.surf.bspline input=ground@lidar raster_output=lake_raleigh_dem_bspline ew_step=6 ns_step=6 method=bicubic lambda_i=0.02 memory=12000

Compute the digital surface model (DSM)

Import lidar data

import las files using v.in.lidar

filter using the first return and classes 1 and 2 

v.in.lidar -t -o input=H:\gisfordesigners\las\0793_009.las output=first_0793_009 return_filter=first class_filter=1,2 --overwrite
v.in.lidar -t -o input=H:\gisfordesigners\las\0793_010.las output=first_0793_010 return_filter=first class_filter=1,2 --overwrite
v.in.lidar -t -o input=H:\gisfordesigners\las\0793_011.las output=first_0793_011 return_filter=first class_filter=1,2 --overwrite
v.in.lidar -t -o input=H:\gisfordesigners\las\0793_013.las output=first_0793_013 return_filter=first class_filter=1,2 --overwrite
v.in.lidar -t -o input=H:\gisfordesigners\las\0793_014.las output=first_0793_014 return_filter=first class_filter=1,2 --overwrite
v.in.lidar -t -o input=H:\gisfordesigners\las\0793_015.las output=first_0793_015 return_filter=first class_filter=1,2 --overwrite
v.in.lidar -t -o input=H:\gisfordesigners\las\0793_017.las output=first_0793_017 return_filter=first class_filter=1,2 --overwrite
v.in.lidar -t -o input=H:\gisfordesigners\las\0793_018.las output=first_0793_018 return_filter=first class_filter=1,2 --overwrite
v.in.lidar -t -o input=H:\gisfordesigners\las\0793_019.las output=first_0793_019 return_filter=first class_filter=1,2 --overwrite

list the imported lidar tiles 

g.list type=vect pattern=first*

combine the lidar tiles with v.patch 

v.patch -b --overwrite v.patch -b --overwrite input=first_0793_009,first_0793_010,first_0793_011,first_0793_013,first_0793_014,first_0793_015,first_0793_017,first_0793_018,first_0793_019 output=first_return

Interpolate the DSM

compute the DSM using the regularized spline with tension interpolation 

v.surf.rst -t --overwrite input=first_return@lidar elevation=lake_raleigh_dsm tension=200 smooth=0.5 npmin=120 dmin=2
