Software

  • Rhinoceros

    Rhino® is a NURBS 3D modeler. Use Rhino 5 on a College of Design computer or download the Windows trial version of Rhino 5 here.


  • Python

    Python is a high-level programming language. Download a python 2.7 distribution with the numpy and scipy packages and the sypder IDE such as Anaconda.


  • Grasshopper

    Grasshopper is an integrated development environment (IDE) for visual programming in Rhino. Download the plugin here.


  • Neon

    Neon is a real-time in viewport renderer for Rhino. Download the plugin here.


  • Keyshot

    Keyshot is an advanced renderer. Download the trial here.
